# 3D Plot Module

QIm's 3D plot module is based on `ImPlot3D`, providing 3D data visualization capabilities.

!!! info "Under Development"
    The 3D plot module is currently under development. Documentation will be added once the features are complete.

## Expected Features

- ✅ 3D scatter plots
- ✅ 3D surface plots
- ✅ 3D wireframe plots
- ✅ 3D axis configuration
- ✅ Mouse interaction (rotation, zoom)
